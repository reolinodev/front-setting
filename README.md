## 목표
parcel-bundler의 기본 세팅입니다.

## 사용법

설치되는 dependancies 입니다.
- parcel-bundler
- sass
- autoprefixer
- postcss
- @babel/core
- @babel/preset-env

설치는 터미널에서 아래와 같이 입력합니다.
<pre>
<code>npm install -D</code>
</pre>

실행은 아래와 같이 입력합니다. 
<pre>
<code>npm run dev</code>
</pre>

배포는 아래와 같이 입력합니다.
<pre>
<code>npm run build</code>
</pre>
